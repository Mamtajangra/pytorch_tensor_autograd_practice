##  What I Learned & Practiced

###  1. Creating Tensors

- Basic math operations (`+`, `-`, `*`, `/`)

- Reshaping: `view()`, `reshape()`

- `.shape`
- `.dtype`
- `.ndim`
- `.device`

###  4. GPU vs CPU Tensors
- Checking availability: `torch.cuda.is_available()`

###  5. Random Tensor Generation
- `torch.rand()`
- `torch.randint()`
- `torch.randn()`
